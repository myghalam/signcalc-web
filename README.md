# SignCalc Web (MVP)

این پروژه را بدون دانش برنامه‌نویسی هم می‌توانید اجرا کنید، فقط مراحل را دقیق انجام دهید.

## مرحله 1) نصب Node.js
نسخه LTS را نصب کنید.

## مرحله 2) قرار دادن فونت‌ها
این دو فایل را داخل مسیر زیر بگذارید:

public/fonts/Vazirmatn-ExtraBold.ttf
public/fonts/Montserrat-Black.ttf

## مرحله 3) اجرا
در پوشه پروژه، این دو دستور را اجرا کنید:

npm install
npm run dev

سپس در مرورگر:
http://localhost:5173
